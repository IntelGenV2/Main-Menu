# Arduino Projects

## Directories

- **Alarm_Clock** - An Arduino-based alarm clock project with time management features
- **Arcade_Games** - Collection of arcade-style games implemented on Arduino hardware
- **Arduino_Tiny_Basic** - A minimal BASIC interpreter for Arduino microcontrollers
- **Dino_Game** - A dinosaur-themed game implementation for Arduino
- **Simon_Says** - The classic Simon Says memory game adapted for Arduino

## Overview

This folder contains various Arduino projects and games demonstrating different capabilities of Arduino microcontrollers.
